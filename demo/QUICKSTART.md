## Troubleshooting

**Icons not showing?**
- You need an internet connection for Font Awesome CDN

**Animations not working?**
- Make sure JavaScript is enabled in your browser
- Try a modern browser (Chrome, Firefox, Safari, Edge)

**Nothing happening when clicking?**
- Check browser console for errors (F12)
- Make sure all files are in the same directory structure
